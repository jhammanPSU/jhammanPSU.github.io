Limitless Apparel

While working at Copyman Print shop one of the clients we helped was wanting to create flyers for the upcoming Disney On Ice event hosted at the Portland Moto Center; I was given multiple images to pick from and a portion of Text that contained various event info.

The client wanted specific info to be visible from a distance along with specifing a vertical lanscape for the overall design.

After looking through the various horizontal images alotted by the client I decided it would be best to use Photoshop to create a viable mesh of multiple images into a vertical format which was then underlayed beneth a layer of information using Indesign to design and format text blocks.
